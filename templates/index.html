About

This project aims to detect and recognize Nepali license plate from an image. Such systems have been successfully implemented in many countries around the world. They are used for various law enforcement purposes like robbery, lost vehicles, criminal chases and vehicles legality. Furthermore, they can be used in automted toll booths and parking areas. Automated license plate reader (ALPRs) systems combine high-speed cameras and sophisticated software to capture and convert license plate images into data that can be compared with information in other databases.License plate reader systems can collect a driverâ€™s geographic location, along with the date and time a vehicle was in a particular place.

How it works

  • The license plate is localized on a vehicle image using CNN. In this project, YOLOv3 a state-of-the-art object detection is used
  • The cropped image is the preprocessed to remove noise, fix distortions and eliminate image arifacts
  • Individual characters are segmented using horizontal and vertical projection
  • Each character is then recognized using an Optical Character Recognition(OCR) engine which is a trained CNN model to predict characters

Note

  • * - The confidence value is for the localization of the plate on the image
  • ** - The predicition time does not include plate localization time.
